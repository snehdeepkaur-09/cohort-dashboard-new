# 📊 Cohort Dashboard

🌐 Live Demo: https://cohort-dashboard-new.vercel.app/

---

## 📌 Overview

The Cohort Dashboard is a data-driven web application designed to visualise and analyse resident cohorts for early intervention insights.

It helps simulate how organisations like Birmingham City Council can move from reactive decision-making to proactive support by identifying at-risk groups using structured data and dashboard analytics.

---

## ✨ Features

- 📊 Interactive cohort dashboard
- 👥 Resident cohort segmentation views
- ⚠️ Risk analysis indicators
- 🧭 Clean multi-page navigation
- 📈 Data-driven UI for insights
- ⚡ Fast client-side routing

---
